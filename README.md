# POV: A New Discovery Mode for Pinterest

An interactive product concept exploring how AI-powered "expert lenses" could transform discovery on Pinterest boards.

**[View Live Demo →](https://pinterest-concept.rachelaubrey.com)**

## Overview

This prototype demonstrates a new discovery mode where users can view new content for their Pinterest boards through different expert perspectives, for example a Dream kitchen board would include suggested pins by an Interior Designer, a Personal Chef, and a Professional organizer. A user could browse through various lenses. The concept addresses a core tension for a visual discovery platform: how AI can raise the quality bar on content without short-cutting a user's desire to browse, curate, and ultimately develop their own taste and sense of style.

Built with vanilla HTML/CSS/JavaScript. Deployed on Netlify.
