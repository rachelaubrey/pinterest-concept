# POV: A New Discovery Mode for Pinterest

An interactive product concept exploring how AI-powered "expert lenses" could transform discovery on Pinterest boards.

**[View Live Demo →](https://pinterest-concept.rachelaubrey.com)**

## Overview

This prototype demonstrates a new discovery mode where users can view their Pinterest boards through different expert perspectives—Interior Designer, Travel Planner, Wedding Planner, and brand-sponsored lenses. The concept addresses a core tension in visual discovery: how AI can enrich the curation process without flattening the user's role in taste formation.

Built with vanilla HTML/CSS/JavaScript. Deployed on Netlify.
